-IFRAME-

Para insertar algún video, documento o algun medio interactivo, usamos (iframe), copiandolo directamente desde el material original

-BACKGROUND IMAGE-

Para insertar una imagen en este formato, crearemos un div para luego usar (background-image: url()) en su CSS, tambíen usando (background-repeat:) para elegir si la imagen se repetirá al terminarse dentro del div y (background-size:) para elegir si queremos que la imagen cubra todo el div o solo se contenga.
