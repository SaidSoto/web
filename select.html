-PSEUDOCLASE/HOVER-

Las pseudoclases se utilizan para hacer referencia a elementos HTML que tengan un cierto comportamiento concreto.

(selector:pseudoclase { propiedad: valor; })

Creando un botón con la etiqueta (button) y usando hover en el style, podemos hacer que este cambie de color cuando pasamos el puntero por encima.

-ATRIBUTO-

El selector de atributos CSS coincide con los elementos en función de la presencia o el valor de un atributo determinado.

EJEMPLO:

(a[href$=".org"]), con este podremos añadir alguna propiedad a cualquier etiqueta (a) en la cual su enlace termine en (.org).

En este caso, usaré (background-color: green) para que el (a) en caso de cumplir la condición tenga fondo verde.
